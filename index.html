几分钟完成接入

兼容 OpenAI SDK 与 REST，以下示例可直接复制运行。

cURL 
curl https://openmix.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-openmix-***" \
  -d '{
    "model": "openai/gpt-4o",
    "messages": [
      {"role": "user", "content": "Hello, OpenMix!"}
    ],
    "stream": true
  }'
Python 
from openai import OpenAI

client = OpenAI(
    base_url="https://openmix.ai/v1",
    api_key="sk-openmix-***",
)

r = client.chat.completions.create(
    model="qwen/qwen-72b",
    messages=[{"role": "user", "content": "Hello!"}],
)
print(r.choices[0].message.content)
